## 控件详细说明

### 1. 关键词 (keyword)

**类型**: text

**位置**: 页面顶部固定

**DOM 特征**:
```
textbox "按职位/公司等搜人才"
```

**交互方式**:
1. 直接在输入框中输入关键词
2. 多个关键词用逗号分隔

---

### 2. 关键词模式 (keywordMode)

**类型**: select (点击切换)

**位置**: 关键词输入框下方

**DOM 特征**:
```
clickable "满足所有关键词搜索"
clickable "所有"
```

**可选值**:
- AND (满足所有关键词) - 默认
- OR (满足任一关键词)

**交互方式**:
1. 点击切换 AND/OR

---

### 3. 城市地区 (city)

**类型**: cascade (搜索 + 级联选择 + 支持多选)

**触发**: 点击 "城市地区" 按钮

**DOM 特征**:
```
clickable "城市地区"
textbox "请输入城市地区"
checkbox "期望"
checkbox "现居"
clickable "历史/热门"
clickable "北京", "上海", "深圳", "杭州", ...
```

**交互方式**:
1. 点击 "城市地区" 展开面板
2. 可选：勾选 "期望" 或 "现居" 复选框
3. **多选模式**：可连续选择多个城市
   - 方式一：点击热门城市标签直接选择
   - 方式二：在搜索框输入城市名称，等待推荐列表，点击匹配项
4. 面板自动关闭（无需点击确定）

**多城市选择流程**:
1. 面板展开后，依次点击或搜索每个城市
2. 每次选择后等待 DOM 更新
3. 所有城市选择完成后，面板会自动关闭

**热门省份/城市**:
- 直辖市: 北京、上海、重庆、天津
- 省份: 广东、浙江、江苏、四川、湖北、安徽、山东、海外...
- 热门城市: 深圳、杭州、广州、成都、南京、武汉、苏州

**重要**:
- 多个城市是独立的点选操作，不需要关闭面板再打开
- 每个城市选择后需要重新获取快照

### 4. 学历要求 (education)

**类型**: select (单选下拉 + 二级选项 + 确定按钮)

**触发**: 点击 "学历要求" 按钮

**DOM 特征**:
```
clickable "学历要求"
clickable "专科及以上"
clickable "本科及以上"
clickable "硕士及以上"
clickable "博士"
clickable "不限"
clickable "确定"
```

**二级选项**（部分选项有子选项）:
| 父选项 | 二级选项 |
|-------|---------|
| 专科及以上 | 专科, 不限 |
| 本科及以上 | 本科, 不限, 只看统招本科 |
| 硕士及以上 | 硕士, 不限 |
| 博士 | 无二级选项 |
| 不限 | 无二级选项 |

**可选值**:
| 显示值 | 说明 |
|-------|------|
| 专科及以上 | 大专学历 |
| 本科及以上 | 本科学历 |
| 硕士及以上 | 硕士学历 |
| 博士 | 博士学历 |
| 不限 | 无学历要求 |

**交互方式**:
1. 点击 "学历要求" 展开下拉
2. 点击父选项（如 "本科及以上"）
3. **检查是否有二级选项**，如果有则点击二级选项
4. **必须**点击 "确定" 按钮

**重要**:
- 点击父选项后需要等待 DOM 更新，重新获取快照
- 二级选项通常是简短文本（"本科"、"不限"等）
- "博士"和"不限"没有二级选项

## 通用交互模式

### 下拉选择控件 (select)

```
1. 点击控件名称按钮展开下拉面板
2. 点击选项选择值
3. (如有) 点击 "确定" 按钮确认
```

**需要点击确定的控件**:
- 学历要求
- 工作年限
- 行业方向

### 搜索+标签控件 (tags)

```
1. 点击控件名称按钮展开面板
2. 可选：勾选复选框 (如 正任职/曾任职)
3. 方式一：在搜索框输入内容
4. 方式二：点击预设标签
5. 可选择多个值
```

**带复选框的控件**:
- 城市地区 (期望/现居)
- 就职公司 (正任职/曾任职)

### 级联选择控件 (cascade)

```
1. 点击控件名称按钮展开面板
2. 可选：在搜索框输入内容
3. 点击一级分类
4. 点击二级分类
5. (如有) 点击 "确定" 按钮确认
```

### 范围选择控件 (range)

```
1. 点击控件名称按钮展开面板
2. 在第一个下拉框选择最小值
3. 在第二个下拉框选择最大值
```

## 注意事项

1. **控件展开后需要重新获取 refs**: 每次点击展开控件后，DOM 会更新，需要重新 snapshot
2. **下拉面板可能重叠**: 同时只能展开一个下拉面板
3. **部分控件需要点击确定**: 学历要求、工作年限、行业方向等需要点击确定按钮
4. **搜索框支持模糊匹配**: 城市地区、就职公司、职位名称等支持搜索
5. **多选控件**: 就职公司、职位名称、行业方向、城市地区支持多选
6. **复选框默认状态**: 城市地区的"期望"和"现居"默认都勾选
7. **搜索建议型控件**: 专业、家乡、毕业学校等输入后必须点选首位推荐项
8. **范围选择需要滚动**: 年龄、期望月薪的下拉选项较多，目标值可能需要滚动查找
9. **学历要求有二级选项**: 本科及以上、硕士及以上等选项有子选项，需要选择到末级

## 常见问题与解决方案

### 1. 关键词填入导航栏搜索框

**问题**: 页面有两个搜索框，直接使用 `fillByPlaceholder` 可能填入导航栏搜索框

**解决**: 使用专用方法 `fillKeywordInput`，通过查找 placeholder 包含"搜人才"的输入框定位

### 2. 学历要求未生效

**问题**: 点击"本科及以上"后直接点确定，未选择二级选项

**解决**: 点击父选项后，重新获取快照，检查是否有二级选项（"本科"、"不限"等），选择后再点确定

### 3. 专业/家乡未生效

**问题**: 输入值后面板关闭，但值未保存

**解决**: 输入后必须点选首位推荐项，然后点击空白区域关闭面板

### 4. 年龄/期望月薪找不到目标值

**问题**: 目标值（如 35岁、40K）在下拉列表中不可见

**解决**: 使用 `selectDropdownOptionWithScroll` 方法，在下拉面板内滚动查找目标值
