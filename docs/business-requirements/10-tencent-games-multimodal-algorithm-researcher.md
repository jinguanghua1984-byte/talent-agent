- 汇报关系：隶属 IEG 团队，汇报给公线技术中台 GM-1 级别算法负责人

## 业务背景

腾讯游戏和腾讯视频希望单独推进视频生成与世界模型探索，围绕游戏世界场景构建更专业的生成式能力。该方向不是替代混元通用大模型，而是与混元协作，面向视频和游戏业务中更高专业度的场景建立专门模型与算法能力。

岗位目标是从 0 到 1 建设生成式游戏引擎相关算法能力，支撑视频生成、视频预测、视频编辑和可交互世界模型等方向的研发与落地。

## 岗位职责

1. 负责生成式游戏引擎相关核心算法的研发与落地，包括视频生成、视频预测、视频编辑等方向。
2. 探索面向游戏世界场景的新一代视频生成技术，构建具备可交互性、可编辑性、物理一致性和时空一致性的世界模型。
3. 结合游戏场景推动生成模型从研究原型到工程实现，形成可被业务使用的模型、工具链或算法模块。
4. 在大规模训练、后训练、推理或数据工程链路中推动多模态模型效果、稳定性和交付效率提升。

## 任职要求

### 硬性要求

1. 教育背景：人工智能、计算机科学、数学或相关专业，硕士及以上学历；博士优先。
2. 编程与工程：熟练掌握 PyTorch 等主流深度学习框架，具备扎实编程能力和工程实现能力。
3. 研究复现与落地：能够高质量复现前沿论文，并推动算法从研究原型进入工程实现。
4. 视频生成或世界模型经验：具备视频生成、视频预测、视频编辑、图像生成、多模态生成或世界模型相关研究/工程实践经验。
5. 至少满足以下技术栈之一：
   - 生成模型：深入理解 Diffusion Models、GANs、VAEs、Flow-based Models 等，具备图像生成或视频生成相关研究或实践经验。
   - 模型训练：有大规模 GPU 集群上的模型预训练、微调或后训练经验；具备多模态大模型训练或应用经验。
   - 多模态模型：具备 CLIP、LLaVA、Qwen 等多模态大模型训练、应用或工程落地经验。

### 加分项

1. 在 SIGGRAPH、CVPR、ICCV、NeurIPS 等顶级会议或期刊发表过相关领域一作论文。
2. 具备游戏、视频、AIGC、可交互生成、可编辑生成、物理一致性或时空一致性建模经验。
3. 有从 0 到 1 建设模型算法方向、带小团队攻关或跨团队推动业务落地经验。
4. 有视频生成产品、世界模型、生成式游戏引擎或多模态大模型平台的实际交付经验。
