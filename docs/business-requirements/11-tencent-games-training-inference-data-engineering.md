- 岗位名称：训练推理数据工程研发专家/工程师

## 业务背景

腾讯游戏和腾讯视频希望独立推进视频生成与世界模型探索，在游戏和视频业务场景中建立更专业的大模型能力。该方向与混元是协作关系，不是替代混元；混元承担更通用的内部业务，而视频和游戏业务对专业度、工程效率、训练推理性能和数据链路有更高要求。

岗位目标是从 0 到 1 搭建生成式游戏引擎相关的大模型训练、推理和数据工程能力，支撑视频生成、视频理解、视频编辑、世界模型、多模态 RLHF 训练与推理平台等方向落地。

## 岗位职责

1. 负责生成式游戏引擎大模型分布式训练和推理系统的性能优化。
2. 负责数据并行、模型并行、流水线并行、专家并行等并行训练策略的工程实现与性能优化。
3. 解决大模型训练中的显存瓶颈、通信延迟、负载均衡等核心计算资源问题。
4. 负责多模态 RLHF 训练与推理平台的实现，支撑训练、推理和数据工程链路落地。
5. 围绕视频生成、视频理解、视频编辑和世界模型业务需求，建设可复用的训练推理平台能力和工程效率体系。

## 任职要求

### 硬性要求

1. 熟悉 GPU 架构、CUDA 编程和算子融合优化，具备训练或推理性能调优经验。
2. 熟悉 PyTorch FSDP、DeepSpeed、Megatron-LM 等分布式训练框架，具备大模型多种并行训练策略的实战经验。
3. 熟悉 vLLM、SGLang 等大模型推理框架，有实际性能调优经验，例如 KV Cache 优化、动态批处理、Attention 算子定制等。
4. 熟悉大模型强化学习相关工程技术，例如 OpenRLHF 或同类 RLHF 训练框架。
5. 具备大模型训练、推理或数据工程方向的工程落地能力，能处理大规模 GPU 集群、平台化训练推理、模型服务性能和稳定性问题。
6. 对视频生成、视频理解、视频编辑、多模态大模型或世界模型相关业务有理解或项目经验优先。

### 加分项

1. 有生成式游戏引擎、视频生成、世界模型、多模态 RLHF 或多模态大模型平台建设经验。
2. 有从 0 到 1 搭建训练、推理、数据工程平台或 AI Infra 能力的经验。
3. 有大规模模型训练集群、推理服务平台、模型并行策略、推理引擎优化或端到端模型交付经验。
4. 有视频生成、视频理解、视频编辑等业务侧算法或平台合作经验。
