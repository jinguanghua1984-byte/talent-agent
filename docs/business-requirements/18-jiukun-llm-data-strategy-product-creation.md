# 九坤至知研究院-大模型数据策略产品经理（创作）

> 来源：飞书 Wiki《大模型产品for 九坤》。本文已过滤脉脉寻访步骤，只保留可用于本地人才库匹配的目标公司、学历要求和岗位能力要求。

## 岗位基本信息

- 岗位名称：大模型数据策略产品经理（创作）
- 用人团队：九坤至知研究院
- 业务方向：故事创作、文案撰写、风格模仿、内容润色和创意写作
- 岗位类型：大模型产品经理、AIGC 产品经理、内容产品经理、创作工具产品经理
- 学历要求：211/985 院校全日制本科及以上学历

## 岗位职责

1. 围绕故事创作、文案撰写、风格模仿、内容润色等任务，评估模型输出的创意性、文笔、节奏与风格一致性。
2. 生产引导模型进行高质量创作的 SFT 数据与系统指令，支持模型在创意生成和内容优化场景中的能力提升。
3. 对供应商产出的文学性、营销文案等创作类数据进行专业验收，把控创意、文法、结构和受众感受。
4. 参与制定小说、诗歌、剧本、评论、营销文案等不同文体的评估维度，优化创意生成与内容优化的标注流程。
5. 分析总结优秀作品的叙事框架、语言风格与情感张力，构建创作领域的 Prompt 设计方法论。

## 任职要求

### 硬性要求

1. 211/985 院校全日制本科及以上学历，汉语言文学、新闻传播、戏剧影视文学、创意写作等专业优先。
2. 具备优秀文笔、叙事能力和审美判断力，能理解文字风格、节奏、结构和受众心理。
3. 有公开出版作品，或在新媒体、出版社、影视机构从事编辑、审校、编剧、编导等实践经验。
4. 具备内容结构规划能力，能把创作质量要求转化为可评估、可标注、可验收的标准。

### 加分项

1. 有 AIGC 创作工具、内容生成产品、文案工具、创作者平台、Prompt 设计或内容质量评测经验。
2. 能提供个人作品集、代表项目或创作类产品落地证据。
3. 对语言敏感，有强成品意识和细节把控力。

## 候选人画像

### 优先人选

1. AIGC 创作工具、内容产品、创作者平台、文本生成产品、内容质量评测方向的产品经理。
2. 具备创作能力、审美判断、Prompt 方法论和数据评估标准建设能力的人选。
3. 能证明创意生成、内容润色、风格控制、文体评估和跨团队推进经验的人选。

## 排除或降权项

1. 只有泛内容运营经验，缺少创作质量判断、产品化标准或大模型数据策略能力。
2. 只有普通产品经验，缺少 AIGC、内容生成、Prompt、模型评估或文本质量体系经验。
3. 只有个人创作爱好，缺少可交付作品、项目证据或可复用方法论。
