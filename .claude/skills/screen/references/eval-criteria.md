# 筛选评估标准

## 评估维度（5个）

### 1. 岗位匹配度（权重30%）

职责对口程度。

评估要素：
- 核心职责是否直接相关（候选人当前/过往职责 vs JD 核心职责）
- 管理层级是否匹配（IC vs 管理者 vs 高管）
- 行业背景是否对口（所在行业与目标行业的重叠度）

评分参考：
| 情况 | 分值范围 |
|------|---------|
| 职责完全一致，管理层级匹配，行业对口 | 85-100 |
| 职责高度相关，层级略有差异 | 65-84 |
| 职责部分重叠，需要转型 | 40-64 |
| 职责差异大，仅少量可迁移技能 | 0-39 |

### 2. 技能覆盖率（权重25%）

JD 要求技能的满足比例。

评估要素：
- 必备技能满足数 / 必备技能总数
- 优先技能满足数 / 优先技能总数
- 技能深度（年限、项目规模、应用场景）

评分参考：
| 情况 | 分值范围 |
|------|---------|
| 必备技能 90%+ 满足，优先技能 70%+ | 85-100 |
| 必备技能 70%+ 满足，优先技能 50%+ | 65-84 |
| 必备技能 50%+ 满足，优先技能 30%+ | 40-64 |
| 必备技能 <50% | 0-39 |

### 3. 经验深度（权重20%）

工作年限和管理经验。

评估要素：
- 总工作年限 vs JD 要求（满足/超出/不足）
- 相关领域经验年限
- 管理经验（人数、层级、范围）

评分参考：
| 情况 | 分值范围 |
|------|---------|
| 超出 JD 要求，管理经验丰富 | 85-100 |
| 满足 JD 要求，有一定管理经验 | 65-84 |
| 低于 JD 要求 1-2 年，管理经验有限 | 40-64 |
| 远低于 JD 要求 | 0-39 |

### 4. 行业背景（权重15%）

相关行业经验。

评估要素：
- 是否在同一行业
- 是否在目标公司类型的竞争对手
- 行业认知深度（从职责描述推断）

评分参考：
| 情况 | 分值范围 |
|------|---------|
| 同行业同类型公司，认知深度足够 | 85-100 |
| 同行业不同类型公司，或相邻行业 | 65-84 |
| 相关行业，部分可迁移 | 40-64 |
| 跨行业，需较长时间适应 | 0-39 |

### 5. 稳定性（权重10%）

职业轨迹稳定性。

评估要素：
- 平均在职时长
- 空窗期频率和长度
- 职业发展轨迹（上升/平移/下降）

评分参考：
| 情况 | 分值范围 |
|------|---------|
| 平均 3 年+/段，轨迹上升，无长空窗 | 85-100 |
| 平均 2-3 年/段，轨迹平稳 | 65-84 |
| 平均 1-2 年/段，或有短期空窗 | 40-64 |
| 平均 <1 年/段，或频繁跳槽 | 0-39 |

---

## 评分规则

- 每个维度 0-100 分
- 加权总分 = round(维度分 * 权重)
- 总分 = sum(各维度加权分)
- 总分范围：0-100，四舍五入取整

权重汇总：
| 维度 | 权重 |
|------|------|
| 岗位匹配度 | 30% |
| 技能覆盖率 | 25% |
| 经验深度 | 20% |
| 行业背景 | 15% |
| 稳定性 | 10% |

---

## 红旗规则（扣分项）

| 红旗 | 触发条件 | 影响 |
|------|---------|------|
| 高流动性 | 3年内跳槽3次+ | 总分-10，标记"高流动性风险" |
| 长空窗期 | 空窗期超过6个月 | 总分-5，标记"需关注空窗原因" |
| 时间重叠 | 工作经历时间有重叠 | 总分-10，标记"经历可能虚报" |
| 学历疑点 | 学历与工作年限不匹配 | 标记"建议学历验证"（不扣分） |
| 方向不符 | 最近职业方向与JD完全不符 | 总分-15，标记"方向偏离" |

红旗检测在基础评分之后执行，扣分直接作用于总分。多个红旗可叠加扣分，但总分不低于 0。

标记格式：在评估报告中以 `:warning:` 图标标注，同时在详细分析中说明原因。
