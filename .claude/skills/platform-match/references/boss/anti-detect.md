# Boss 直聘反检测策略

> 状态: 已校准（2026-04-20）

## 已确认

- **`page.evaluate(fetch)` 会触发反爬**: 调用后页面强制登出（API 返回 code: 7）
- **被动拦截安全**: 使用 `ctx.on('response')` 监听 API 响应不会触发反爬

## 频率控制

| 参数 | CDP 模式 | Headless 模式 |
|------|---------|-------------|
| 单次搜索间隔 | 5-10s | 10-20s |
| 单批搜索上限 | 20 | 10 |
| 每日搜索上限 | 150 | 60 |
| 连续操作上限 | 30 分钟暂停 60-120s | 20 分钟暂停 60-120s |
| 单页翻页间隔 | 2-5s | 5-10s |

## 推荐方案

1. 使用被动网络拦截（`page.on('response')` 或 `ctx.on('response')`）
2. 用户手动在浏览器中执行搜索，脚本监听并捕获响应
3. 请求间隔随机化（非固定间隔）
4. 单次搜索翻页不超过 3 页
5. 连续搜索 30 分钟后暂停 60-120s
