# 脉脉字段映射参考

本文档提供脉脉人才搜索页面的字段映射参考，用于将用户输入映射到页面控件。

> **注意**：脉脉页面可能会更新，以下选择器和结构仅供参考。实际使用时需要根据当前页面结构动态调整。

## 搜索条件控件

### 1. 职位名称（关键词搜索）

```
控件类型：文本输入框
页面位置：搜索栏主输入框
操作方式：
  - 点击搜索框聚焦
  - 输入职位关键词
  - 按 Enter 或点击搜索按钮
```

### 2. 城市地区

```
控件类型：下拉多选
页面位置：筛选条件区域
操作方式：
  - 点击"城市"下拉框
  - 从列表中选择城市（支持多选）
  - 点击外部区域关闭下拉框
```

### 3. 学历要求

```
控件类型：下拉单选
页面位置：筛选条件区域
选项值：
  - 不限
  - 大专
  - 本科
  - 硕士
  - 博士
操作方式：
  - 点击"学历"下拉框
  - 选择对应选项
```

### 4. 工作年限

```
控件类型：下拉单选 / 范围选择
页面位置：筛选条件区域
选项值：
  - 不限
  - 1年以下
  - 1-3年
  - 3-5年
  - 5-10年
  - 10年以上
操作方式：
  - 点击"工作年限"下拉框
  - 选择对应选项
```

### 5. 就职公司

```
控件类型：文本输入 + 智能提示
页面位置：高级筛选区域
操作方式：
  - 点击"公司"输入框
  - 输入公司名称
  - 从智能提示中选择匹配项
```

### 6. 行业方向

```
控件类型：下拉多选
页面位置：筛选条件区域
常见选项：
  - 互联网/IT
  - 金融
  - 教育
  - 医疗健康
  - 电商
  - 企业服务
  - ...
操作方式：
  - 点击"行业"下拉框
  - 选择行业标签
```

### 7. 毕业学校

```
控件类型：文本输入 + 智能提示
页面位置：高级筛选区域
操作方式：
  - 点击"学校"输入框
  - 输入学校名称
  - 从提示中选择
```

### 8. 专业

```
控件类型：文本输入
页面位置：高级筛选区域
操作方式：
  - 点击"专业"输入框
  - 输入专业名称
```

### 9. 性别

```
控件类型：单选按钮组 / 下拉单选
页面位置：筛选条件区域
选项值：
  - 不限
  - 男
  - 女
```

### 10. 年龄

```
控件类型：范围选择器
页面位置：高级筛选区域
输入格式：最小年龄 - 最大年龄
示例：25-35
```

### 11. 期望月薪

```
控件类型：范围选择器 / 下拉单选
页面位置：筛选条件区域
选项值：
  - 不限
  - 10K以下
  - 10-20K
  - 20-30K
  - 30-50K
  - 50K以上
```

### 12. 家乡

```
控件类型：下拉选择
页面位置：高级筛选区域
操作方式：
  - 点击"家乡"下拉框
  - 选择省/市
```

## 候选人列表字段

### 列表页卡片结构

每个候选人卡片通常包含以下信息：

| 字段 | 选择器参考 | 说明 |
|------|-----------|------|
| 姓名 | `.candidate-name` / `[data-name]` | 候选人姓名 |
| 头像 | `.avatar` / `img` | 头像图片 |
| 活跃度 | `.active-status` / `.online` | 活跃状态标签 |
| 当前职位 | `.position` / `.job-title` | 当前职位名称 |
| 当前公司 | `.company` / `.employer` | 当前就职公司 |
| 工作年限 | `.experience` / `.work-years` | 工作经验 |
| 学历 | `.education` / `.degree` | 最高学历 |
| 工作地点 | `.location` / `.city` | 当前工作城市 |
| 期望薪资 | `.salary` / `.expected-salary` | 期望薪资范围 |

### 详情页字段

点击候选人卡片后，详情页/弹窗包含完整信息：

| 字段 | 位置 | 说明 |
|------|------|------|
| 职业标签 | 个人信息区域 | 多个技能/标签 |
| 期望职位 | 求职意向区域 | 期望的职位类型 |
| 期望工作地点 | 求职意向区域 | 期望工作城市 |
| 工作履历 | 工作经历区域 | 多段工作经历 |
| 教育履历 | 教育经历区域 | 多段教育经历 |

### 工作履历结构

```json
{
  "workHistory": [
    {
      "timeRange": "2022.03 - 至今",
      "company": "某互联网公司",
      "position": "高级Java开发工程师",
      "description": "负责核心业务系统开发..."
    },
    {
      "timeRange": "2019.06 - 2022.02",
      "company": "某科技公司",
      "position": "Java开发工程师",
      "description": "参与电商平台开发..."
    }
  ]
}
```

### 教育履历结构

```json
{
  "educationHistory": [
    {
      "timeRange": "2015.09 - 2019.06",
      "school": "某某大学",
      "major": "计算机科学与技术",

    }
  ]

```

## 分页控件

```
位置：列表底部
类型：页码导航 / 加载更多按钮

操作方式：
  - 检测"下一页"按钮是否存在且可点击
  - 点击翻页后等待新内容加载
  - 检测是否到达最后一页（按钮禁用或消失）
```

## 详情页打开方式

```
方式1：点击卡片整体 -> 弹窗显示详情
方式2：点击卡片 -> 新页面/路由跳转

关闭方式：
  - 弹窗：点击关闭按钮或外部区域
  - 新页面：关闭标签页或返回上一页
```

## 动态适配建议

由于脉脉页面可能更新，建议采取以下策略：

1. **首次访问时探测**：使用 agent-browser 的截图和 DOM 分析能力，识别当前页面结构
2. **关键词匹配**：通过文本内容（如"城市"、"学历"）定位控件，而非固定选择器
3. **容错处理**：如果某个控件找不到，跳过该条件并告知用户
4. **截图确认**：关键步骤截图，让用户确认操作正确
